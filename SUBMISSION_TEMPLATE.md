## Project Description
A powerful Python tool that converts blog posts and articles to clean markdown using the Firecrawl SDK. Features include command-line interface, batch processing, customizable output options, and comprehensive error handling.

## Key Features
✅ Convert any blog post URL to clean markdown
✅ Command-line interface with multiple options
✅ Batch processing for multiple URLs
✅ Customizable scrape options via JSON
✅ Error handling and retry logic
✅ Clean, human-readable output
✅ Comprehensive documentation and examples

- **Architecture**: Modular design with examples and test suite

## Usage Examples

### Basic Conversion
```bash
python blog_converter.py -u "https://example.com/blog-post"
```

### Custom Output Directory
```bash
python blog_converter.py -u "https://example.com/blog-post" -o "converted_posts"
```

### Batch Processing
```bash
python examples/batch_converter.py
```
