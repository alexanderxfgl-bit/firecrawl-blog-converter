# Firecrawl Blog to Markdown Converter

A powerful Python tool that converts blog posts and articles to clean markdown using the Firecrawl SDK. Perfect for content creators, researchers, and anyone who needs to convert web content to markdown format.

## Features

- 🚀 Convert any blog post URL to clean markdown
- 📝 Output formatted markdown with metadata headers
- 🔧 Configurable scrape options via JSON
- 💾 Custom output directories
- 📊 Word count and preview functionality
- ⚡ Fast and efficient web scraping

## Installation

1. Clone this repository:
```bash
git clone <repository-url>
cd firecrawl-blog-converter
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Set up your Firecrawl API key:
```bash
export FIRECRAWL_API_KEY="your-api-key-here"
```

## Usage

### Command Line Interface

Basic usage:
```bash
python blog_converter.py -u "https://example.com/blog-post"
```

With custom output directory:
```bash
python blog_converter.py -u "https://example.com/blog-post" -o "my_articles"
```

As text format instead of markdown:
```bash
python blog_converter.py -u "https://example.com/blog-post" -f "text"
```

With custom scrape options:
```bash
python blog_converter.py -u "https://example.com/blog-post" -s '{"includeTags": ["main"], "excludeTags": ["nav", "footer"]}'
```

### Programmatic Usage

```python
from blog_converter import convert_blog

# Convert a blog post
convert_blog(url="https://example.com/blog-post")
```

## Examples

### Example 1: Convert a blog post
```bash
python blog_converter.py -u "https://medium.com/@example/my-great-article"
```

### Example 2: Batch convert multiple blogs
```bash
urls=(
    "https://blog.example.com/post1"
    "https://blog.example.com/post2"
    "https://blog.example.com/post3"
)

for url in "${urls[@]}"; do
    python blog_converter.py -u "$url" -o "converted_posts"
done
```

### Example 3: Convert with custom options
```bash
python blog_converter.py -u "https://news.example.com/article" -s '{"waitFor": 2000, "onlyMainContent": true}'
```

## API Configuration

You need a Firecrawl API key to use this tool. Get your API key from:

1. [Firecrawl Dashboard](https://firecrawl.com/dashboard)
2. Or sign up at [Firecrawl Website](https://firecrawl.com)

Set the environment variable:
```bash
export FIRECRAWL_API_KEY="your-api-key-here"
```

## Output Format

The converter creates markdown files with structured headers:

```markdown
# Blog Post Converted to Markdown

*Generated by Firecrawl Blog Converter*
*URL: https://example.com/blog-post*

---

[Clean markdown content from the blog post...]
```
